# Vite
A simple and mnml static site generator.

## Usage
```console
$ vite.py new <project-path>
```
Write your Markdown files in `pages/` and  
```console
$ vite.py build <project-path>
```

*OR*

```console
$ make.py  # at the project directory
```
Rendered HTML will be in `build/`
